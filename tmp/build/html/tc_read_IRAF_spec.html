tc_read_IRAF_spec.wave_1dspec(header)¶

Return type:np.array
tc_read_IRAF_spec.wave_multispec(header)¶

calculate wavelength array from MULTISPEC header

This function extracts information from the header of a fits file that contains spectral data. From this information, the wavelength array is calculated.

..note: While some checking on the fits header keywords is done, the
implementation is not complete. Also, many telescopes use non-standard keywords or do not write required values. Be wary!
Parameters:header (astropy.io.fits.header.Header) – header of fits file
Returns:wave – wavelength array in the format specified in the header. Depending on the header keywords, this can be one or two dimensional.
Return type:np.array
